# orientacao-objetos-php
PHP Orientado a Objetos (POO) é um paradigma fundamental no desenvolvimento PHP moderno (versões 5+), estruturando o código em classes e objetos para melhorar a organização, manutenção e reutilização. Baseia-se em conceitos como encapsulamento, herança, polimorfismo e classes, e é indispensável para o uso de frameworks e boa engenharia de software.
